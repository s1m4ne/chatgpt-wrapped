# Product Overview

ChatGPT Wrappedは、ユーザーのChatGPT利用履歴を可視化・分析するWebアプリケーションです。Spotify Wrappedにインスパイアされ、1年間のAI対話パターンを美しく振り返ることができます。

## Core Capabilities

- **会話データの解析**: ChatGPTエクスポート(conversations.json)をパースし、正規化された内部形式に変換
- **統計分析**: 利用頻度、アクティビティパターン、セッション傾向を自動計算
- **LLM駆動インサイト**: Gemini APIを活用した深層分析（トピック分類、文体診断、ベスト発言抽出）
- **カード形式表示**: スワイプ可能なカードUIで分析結果を順次表示
- **シェア機能**: 分析結果を画像としてエクスポート

## Target Use Cases

- ChatGPTヘビーユーザーが自分の利用傾向を把握したい
- 年末の振り返りとしてAI対話の統計を可視化したい
- どのようなトピックでAIを活用しているか俯瞰したい

## Value Proposition

- プライバシー重視: データはブラウザ内で処理、サーバー送信なし
- 即時フィードバック: ファイルアップロード後、基本統計は即座に表示
- LLM活用: 単純な統計を超えた、意味的な分析をGemini APIで実現
